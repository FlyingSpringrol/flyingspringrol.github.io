PantheonVR

Pantheon was originally concieved as a 3D modeling tool for Virtual Reality. We thought it made more sense to build things for VR in VR. So some friends and I built a tool for it.

We did YCombinator. I won't ever forget that sense of belonging somewhere. I was in charge of marketing, content creation, UI design, and UI code.
I think this idea will exist in ten years, when VR is finally ready for consumers.
